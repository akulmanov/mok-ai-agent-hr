# Telegram Bot для HR-системы отбора

## Описание

Telegram бот для автоматизированного отбора кандидатов. Кандидаты могут отправлять резюме через бота, система автоматически проверяет совместимость с открытыми вакансиями и задает уточняющие вопросы при необходимости.

## Возможности

### Основные функции:
- 📄 **Загрузка резюме** - текстом или файлом (PDF, DOCX, TXT)
- 🔍 **Автоматическая проверка совместимости** - с открытыми вакансиями
- ❓ **Диалог с уточняющими вопросами** - система задает вопросы и обрабатывает ответы
- 💼 **Просмотр вакансий** - список открытых вакансий с деталями
- 📊 **Статистика кандидата** - результаты всех отборов

### Дополнительные возможности:
- Просмотр деталей вакансий
- Проверка совместимости с конкретной вакансией
- История всех отборов
- Автоматическое обновление профиля с Telegram данными

## Установка

### 1. Создать бота через BotFather

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Сохраните полученный токен

### 2. Настроить переменные окружения

Добавьте в файл `.env`:

```env
TELEGRAM_BOT_TOKEN=your_bot_token_here
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

## Запуск

### Вариант 1: Отдельный процесс

```bash
python run_telegram_bot.py
```

### Вариант 2: Интеграция с основным приложением

Добавьте в `app/main.py`:

```python
from app.telegram_bot import start_bot
import asyncio

# В функции startup
@app.on_event("startup")
async def startup_event():
    # Запустить бота в фоне
    asyncio.create_task(start_bot())
```

### Вариант 3: Docker Compose

Добавьте в `docker-compose.yml`:

```yaml
  telegram-bot:
    build: .
    container_name: hr-screening-telegram-bot
    environment:
      - TELEGRAM_BOT_TOKEN=${TELEGRAM_BOT_TOKEN}
      - DATABASE_URL=${DATABASE_URL:-postgresql://postgres:postgres@postgres:5432/hr_screening}
      - OPENAI_API_KEY=${OPENAI_API_KEY}
    volumes:
      - ./data:/app/data
      - ./uploads:/app/uploads
    command: python run_telegram_bot.py
    depends_on:
      - api
    restart: unless-stopped
    networks:
      - hr-screening-network
```

## Использование

### Команды бота:

- `/start` - Начать работу с ботом
- `/help` - Показать справку
- `/cv` - Загрузить резюме
- `/positions` - Показать открытые вакансии
- `/stats` - Моя статистика
- `/cancel` - Отменить текущую операцию

### Процесс работы:

1. **Загрузка резюме:**
   - Отправьте команду `/cv` или просто отправьте резюме текстом/файлом
   - Система обработает резюме и создаст профиль кандидата

2. **Проверка совместимости:**
   - После загрузки резюме система автоматически проверит совместимость со всеми открытыми вакансиями
   - Покажет топ-3 наиболее подходящих вакансий

3. **Уточняющие вопросы:**
   - Если система нуждается в дополнительной информации, она задаст вопросы
   - Отвечайте на вопросы текстом
   - После ответа на все вопросы система переоценит кандидатуру

4. **Просмотр результатов:**
   - Используйте `/stats` для просмотра всех результатов отбора
   - Используйте `/positions` для просмотра вакансий и проверки совместимости

## Архитектура

### Компоненты:

- **`app/telegram_bot.py`** - Основной модуль бота с обработчиками
- **`run_telegram_bot.py`** - Скрипт для запуска бота
- **User Sessions** - Кэш для хранения состояния пользователей в памяти

### Интеграция с системой:

- Использует `AgentTools` для создания кандидатов
- Использует `TrueAgent` для проверки совместимости
- Сохраняет данные в ту же базу данных, что и веб-интерфейс
- Использует те же модели и схемы данных

## Безопасность

- Токен бота хранится в переменных окружения
- Сессии пользователей хранятся в памяти (можно перенести в Redis для продакшена)
- Валидация файлов перед обработкой
- Обработка ошибок с информативными сообщениями

## Troubleshooting

### Бот не отвечает:
- Проверьте, что токен установлен правильно
- Проверьте логи на наличие ошибок
- Убедитесь, что бот запущен

### Ошибки при обработке файлов:
- Проверьте формат файла (PDF, DOCX, TXT)
- Убедитесь, что файл не поврежден
- Проверьте размер файла (лимит Telegram: 50MB)

### Ошибки базы данных:
- Убедитесь, что база данных доступна
- Проверьте права доступа к файлам базы данных
- Проверьте логи на наличие ошибок SQL
